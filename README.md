# Sidebar Resource Saver

Automatically suspends unused webviews and iframes in collapsed sidebars to drastically save RAM and CPU usage.

## Features
- Suspends webviews (e.g. from the Surfing plugin) when the sidebar is collapsed.
- Restores webviews when the sidebar is expanded.
- Zero configuration required. Works out of the box.
- Prevents error overlays when restoring webviews.

## Installation
1. Search for "Sidebar Resource Saver" in the Obsidian Community Plugins.
2. Install and enable the plugin.

## Compatibility
Requires Obsidian v0.15.0 or higher.
